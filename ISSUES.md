# Proposed Issues for Extracurricular Feature Expansion

This file lists proposed feature work items based on comparison with the external extracurricular management system.

## 1. Add user registration, authentication, and roles
- Student registration with profile fields
- Email activation workflow
- Student login
- Staff/admin login
- Account status approval/management

## 2. Add persistent event management
- Event creation and editing
- Event categories and event types
- Event status lifecycle (Active / Rejected)
- Event details including banner, venue, description, rules, schedule
- Course/class/department/club eligibility filtering

## 3. Add participation workflows
- Student event enrollment
- Team event support with team creation
- Event participation cutoff / capacity enforcement
- Duplicate application prevention
- Participation status tracking

## 4. Add points and ranking
- Points settings for placements and participation
- Student points tally
- Leaderboard / points table
- Rank display for students

## 5. Add complaint management
- Student complaint submission
- Complaint attachments
- Complaint status and staff review

## 6. Add student dashboard
- Personalized dashboard view
- Upcoming participation summary
- Recent result summary
- Team token code generation

## 7. Add admin/staff dashboard and reporting
- Summary counts for students, staff, events, departments
- Quick access to event, results, and complaint management
- Reports for event participation and results
